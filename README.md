   ```bash

   cd tuweb_mobile
   ```

   - Follow step-by-step instructions
   - Kerjakan latihan mandiri untuk pemahaman lebih dalam

   - Simpan semua project yang dibuat
   - Dokumentasikan di GitHub
   - Tambahkan ke CV/portfolio

### Untuk Tutor

1. **Persiapan sebelum kelas**
   - Install semua tools (Node.js, Ionic CLI, Android Studio)
   - Setup emulator Android yang sudah running
   - Siapkan backup code untuk troubleshooting

2. **Saat tutorial**
   - Live coding sambil menjelaskan konsep
   - Pause untuk troubleshooting
   - Encourage mahasiswa untuk experiment

3. **Evaluasi**
   - Gunakan checklist yang disediakan di setiap materi
   - Review code quality, bukan hanya functionality
   - Berikan feedback konstruktif
